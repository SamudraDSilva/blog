### Start Here (First Time)

If this is your first time, read the documents IN THIS ORDER:

Problem Statement (5 min)
What problem you're solving and why

Requirements (15 min)
What features you're building (user stories)

System Architecture (20 min)
High-level overview of how everything connects

Database Schema (15 min)
Database tables and relationships

API Design (20 min)
All API endpoints and how they work

Local Setup Guide (30 min + setup time)
Step-by-step guide to get started coding

Total reading time: ~2 hours
Then: Start coding!
